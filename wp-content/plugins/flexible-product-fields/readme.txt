== Description ==

= Powerful WooCommerce Product Fields Manager =

Customize and personalize your products with the power of WooCommerce product fields and add-ons. Enable customers to customize products with additional options such as text fields, checkboxes, radio buttons or dropdowns.

Customers will be able to add engravings, messages, insurance, gift packaging. Right on the product screen, directly above or below the add to cart button.

= Powerful, yet Simple Fields Editor =

Simplicity is key. There are many bloated fields plugins. We decided to use a simple design as much as possible to create a really flexible fields editor.

= Flexible Placement Options =

Show field groups above or under the add to cart button. Assign to all products, selected products or selected categories. It's your choice.

= Frontend Display =

The fields will seamlessly integrate with your theme with minimum CSS added by the plugin. Apart from just selecting options the plugin will also display a clear options prices (if added) and totals.

= All Features =

* Add field groups above or below WooCommerce add to cart button.
* Assign field groups to selected products.
* Assign field groups to all products or selected categories (PRO).
* Required or optional fields.
* Manage labels, values, placeholders and CSS classes for each field.
* Add fixed or percentage prices to fields (PRO).
* Support for WPML and Polylang.
* Save field groups as drafts (handy when you want to hide field group for some time without deleting it).
* Duplicate field groups.

== Installation	 ==

You can install this plugin like any other WordPress plugin.

1. Download and unzip the latest release zip file.
2. Upload the entire plugin directory to your /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress Administration.

You can also use WordPress uploader to upload plugin zip file in menu Plugins -> Add New -> Upload Plugin. Then go directly to point 3.
